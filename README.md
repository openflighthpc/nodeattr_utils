# NodeattrUtils Library

A Ruby library for handling nodeattr-style host specifications.

## Installation

Add this line to your application's Gemfile:

```
gem 'nodeattr_utils'
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install nodeattr_utils
```
